# ScreenHunter Pro

## Overview

ScreenHunter Pro is a screen capture software designed for Windows users. It enables the capture of images and videos from the desktop, active windows, or specific regions. The tool supports various capture modes to assist in documentation, tutorials, and daily workflow tasks.

## Key Features

**Multiple capture modes including full screen, window, and region**  
**Support for video recording with audio**  
**Automatic file naming and saving options**  
**Built-in image editing tools**  
**Scheduled capture capabilities**  
**Supports various image and video formats**  
**Customizable hotkeys for quick access**  
**Capture cursor and annotations for clarity**

## Why ScreenHunter Pro?

ScreenHunter Pro is chosen for its consistent performance in screen capturing tasks. It offers clear and precise image and video outputs with straightforward usability.
